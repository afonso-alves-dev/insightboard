## 📌 Sobre o projeto

O **InsightBoard** é uma aplicação fullstack desenvolvida para centralizar informações e fornecer insights visuais de forma clara e eficiente.

O objetivo do projeto é oferecer uma experiência intuitiva para usuários que precisam organizar, acompanhar e interpretar dados.

### 🎯 Principais objetivos

* Centralização de dados
* Visualização clara e objetiva
* Interface intuitiva
* Arquitetura escalável

---

## 🧠 Funcionalidades

* 📊 Visualização de dados em dashboard
* ➕ Criação e gerenciamento de registros
* 🔄 Atualização de dados em tempo real *(ou futura implementação)*
* 🗑️ Remoção de registros
* 🔍 Filtros e organização de informações

## 🔌 API (exemplo)

### Criar registro

```http
POST /items
```

**Body:**

```json
{
  "title": "Novo dado",
  "value": 100
}
```

---

### Listar registros

```http
GET /items
```

---

### Atualizar registro

```http
PUT /items/:id
```

---

### Remover registro

```http
DELETE /items/:id
```

## ▶️ Como rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/insightboard.git
```

### 2. Instale as dependências

```bash
cd backend
npm install
```

### 3. Inicie o servidor

```bash
node app.js
```

### 4. Rode o frontend

Abra o arquivo `index.html` no navegador.
